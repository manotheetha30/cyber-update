## Executive Summary

A malicious npm package named postcss-minify-selector-parser masqueraded as a legitimate tool, leading to a Windows RAT with capabilities including remote shell, file transfer, persistence, and Chrome credential theft. The malware used a C2 server at hxxp[:]//95[.]216[.]92[.]207:8080 and leveraged PowerShell and Python-based payloads for execution and persistence.
